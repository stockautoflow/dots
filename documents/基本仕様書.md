# スマートドッツ（Smart Dots）

## 基本仕様書

### 1. 画面レイアウト・UIデザイン仕様

本アプリはWebのシングルページアプリケーション（SPA）として、HTML内の `div` 要素（コンポーネント）の非表示・表示（`.active` クラスの着脱）によって画面遷移を実現する。

#### 1.1 ホーム画面（保護者向けダッシュボード）

* **ヘッダー**：アプリ名「スマートドッツ / Smart Dots」
* **メインエリア**：
* **「今日のレッスンをはじめる / Start Today's Lesson」ボタン**（現在の累計日齢 `current_day` に応じたレッスンが起動。最も目立たせること）。
* **言語設定（Language）トグルスイッチ/セレクトボックス**：[日本語] [English] [Bilingual（日+英）] の3択。


* **サブエリア**：進捗カレンダー（当月分）。完了した日付のマスに星マークが表示される。

#### 1.2 カウントダウン画面

* **UI要素**：背景は白。画面中央に巨大な数字（3 ➔ 2 ➔ 1）が1秒ごとにアニメーション（フェードイン・アウト）表示される。
* **音声**：言語設定に関わらず共通の効果音（ポン、ポン、ポン）または「Ready...」の音声を再生。

#### 1.3 レッスン画面（子ども向けフラッシュ・クイズ・数式共通共通）

子どもの視線逸脱を防ぐため、ナビゲーションバーや広告、ボタン類は一切配置しない。

* **上部マージンエリア**：右隅に小さく「✕ 中断 (Hold 2s)」ボタンのみを配置。
* **中央エリア**：HTML5 `<canvas>` 要素（アスペクト比 `4:3`、最大横幅 `800px` ）。すべてのドッツ、数字、数式、クイズの選択肢はこのCanvas内に描画される。
* **下部テキストエリア**：第3サイクル（数字・数式リンク）時に、対応する文字（例：「5 / Five」や「2 + 3 = 5」）を表示する領域。

#### 1.4 チャイルドロック画面（モーダル）

* 「✕ 中断」ボタンを2秒以上長押しした際、レッスン画面の前面にオーバーレイ表示される。
* **UI要素**：
* 「おうちのひととやってね / For Parents」という警告文。
* ランダムな算数クイズ（例：「6 + 3 = ?」）。
* 0〜9のテンキーボタン（子どもが適当に連打して突破できないよう、キーボードではなく独自の小さな配列を推奨）。
* 「もどる / Cancel」ボタン。

### 2. データ構造・構造体定義（LocalStorage仕様）

ブラウザのLocalStorageに保存するJSONオブジェクトのキー名は `smartdots_v1_data` とし、以下の構造を持つ。

```json
{
  "settings": {
    "language": "bilingual", // "ja" | "en" | "bilingual"
    "speed": 600,             // ミリ秒単位 (500 | 600 | 800)
    "skin": "dynamic"         // "red_circle" | "dynamic"
  },
  "progress": {
    "current_day": 31,        // 現在の日齢（1〜90）
    "history": [
      "2026-06-24",
      "2026-06-25"
    ]
  }
}

```

### 3. イベント・遷移制御仕様

各状態（State）における画面の遷移条件と、Vanilla JSで制御するイベントリスナーを定義する。

#### 3.1 画面遷移マトリクス

| 遷移前画面 | トリガーイベント | 遷移後画面 | 内部処理 |
| --- | --- | --- | --- |
| ホーム | 「今日のレッスン」タップ | カウントダウン | `state.current_day` から本日のデータセットをロード |
| カウントダウン | 3秒経過（タイマー終了） | レッスン（フラッシュ） | `audio.js` のプリロード完了を確認し、ループタイマー起動 |
| レッスン | カリキュラム全データの消化 | 終了演出 | `state.progress.history` に今日の日付を追加、LocalStorage更新 |
| レッスン | 中断ボタンを2秒長押し | チャイルドロック | フラッシュのタイマーおよび音声を一時停止（Pause） |
| チャイルドロック | クイズ正解、または「もどる」 | ホーム、またはレッスン | 正解ならホームへ、もどるならタイマーを再開（Resume） |
| 終了演出 | 「ホームへ」タップ、または5秒放置 | ホーム | `state.current_day` を1インクリメント（翌日へ） |

#### 3.2 タッチイベントの共通制御（0歳児対策）

レッスン画面表示中、ブラウザ標準のジェスチャーを完全にオーバーライドする。

```javascript
// app.js または共通初期化処理での実装イメージ
const preventFormations = () => {
    // 1. ダブルタップズームの禁止
    document.addEventListener('touchstart', (e) => {
        if (e.touches.length > 1) {
            e.preventDefault();
        }
    }, { passive: false });

    // 2. コンテキストメニュー（長押しメニュー）の禁止
    document.addEventListener('contextmenu', (e) => {
        e.preventDefault();
    });
};

```

### 4. 言語別（日本語・英語）リソース定義

第2サイクルのクイズテキストおよび第3サイクルのテキスト表示で使用する、言語リソースマッピング。

```javascript
const LanguageResources = {
    ja: {
        quiz_text: (num) => `どっちが ${num} かな？`,
        congrats: "よくできました！",
        lock_text: "おうちのひととやってね"
    },
    en: {
        quiz_text: (num) => `Which one is ${NumberToWord(num)}?`,
        congrats: "Great job!",
        lock_text: "For Parents Only"
    }
};

// 数字を英語のスペルに変換するヘルパー辞書（1〜50まで保持）
const NumberWords = {
    1: "One", 2: "Two", 3: "Three", 4: "Four", 5: "Five",
    // ... 中略 ...
    20: "Twenty", 30: "Thirty", 40: "Forty", 50: "Fifty"
};
function NumberToWord(num) { return NumberWords[num] || num.toString(); }

```
